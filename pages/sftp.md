- Connect to **sftp**
	- normal connection:
		- `sftp vertis_sftp@secondary-web`
	- connection with bandwidth limitation:
		- `sftp -l 10 vertis_sftp@secondary-web`
- Local working directory
	- `lpwd`
- Remote working directory
	- `pwd`

	- downloads it to the local working directory
		- get <path>
